6 CONNECTED EXIT GAME
TEAM NAME
Now make sure you come up with a catchy team name that's between 5 and 15 characters. This is also shown on the leaderboard, if you make it.
Only one person in your team needs to enter the team name. Once you start the game, the team name can no longer be changed.
GET STARTED
Make sure you've set up a video call and everyone has joined, in case you're not physical together. You can use Zoom, Teams, The Pub Hub or any other tool you'd like.
When all players are ready, any player can check all the checkboxes and press the Start Game button. This will start the game for everyone.
